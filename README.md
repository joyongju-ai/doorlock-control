# doorlock-control

# STM32 기반 도어락 제어 시스템

## 프로젝트 개요
STM32를 활용한 비밀번호 입력 방식의 도어락 시스템입니다. 4x4 키패드, 7-세그먼트 디스플레이, 스피커 모듈을 연동해 구현했습니다.

## 개발 환경
- MCU : STM32H503
- IDE : STM32CubeIDE
- 언어 : C

## 주요 기능
- 4x4 키패드를 통한 비밀번호 입력
- 7-세그먼트 디스플레이로 입력 숫자 실시간 표시
- TIM3 인터럽트 기반 세그먼트 새로고침
- 비밀번호 일치 시 성공음, 불일치 시 실패음 출력
- 3초 입력 타임아웃 자동 초기화
- HAL_GetTick 기반 논블로킹 타임아웃 처리

## 핵심 설계 원칙
TIM3 인터럽트와 메인 루프의 타이밍 충돌 문제를 타이밍 차트로 시각화해 분석하고, 논블로킹 구조로 개선해 다중 입력에도 오작동하지 않는 안정적인 시스템을 구현했습니다.
